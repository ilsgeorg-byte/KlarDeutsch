# 🔍 AI Проверка слов

## 📋 Описание

Скрипт проверяет слова в базе данных на:
- ✅ Правильность написания (немецкий)
- ✅ Правильность перевода (русский)
- ✅ Артикли (der/die/das)
- ✅ Формы глаголов

Использует **Groq API** (бесплатно, быстро).

---

## 🚀 Настройка

### 1. Получи Groq API Key

1. Зайди на https://console.groq.com/keys
2. Создай аккаунт
3. Создай новый API Key
4. Скопируй ключ

### 2. Добавь в `.env.local`

```env
GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxxxxxx
```

### 3. Установи зависимости

```bash
cd api
pip install openai
```

---

## 🔧 Использование

### Режим проверки (без исправлений)

```bash
cd api
python check_words_ai.py
```

Скрипт проверит **100 слов** и покажет ошибки.

---

### Режим исправления

Открой `check_words_ai.py` и измени:

```python
DRY_RUN = False  # Было: True
```

Затем запусти:

```bash
python check_words_ai.py
```

Скрипт **исправит ошибки в БД**.

## ⚙️ Настройки

В начале файла `check_words_ai.py`:

```python
CHECK_LIMIT = 100  # Сколько слов проверить за раз
DRY_RUN = True     # True = проверка, False = исправление
```

---

## 🔄 Массовая проверка

Для проверки **всей базы**:

```python
CHECK_LIMIT = 10000  # Все слова
```

Или запусти несколько раз:

```bash
python check_words_ai.py  # Первые 100
python check_words_ai.py  # Следующие 100
# и т.д.
```

---

## 📝 Что исправляет ИИ

| Тип ошибки | Пример | Исправление |
|------------|--------|-------------|
| **Написание** | "Apfel" → "Apfel" | ✅ Верно |
| **Написание** | "Vater" → "Fater" | ❌ → "Vater" |
| **Перевод** | "Haus" → "Здание" | ❌ → "Дом" |
| **Артикль** | "Mädchen" → der | ❌ → "das" |
| **Глаголы** | "gehen" → ging, gegangen | ❌ → "gehen, ging, ist gegangen" |

---

## 💡 Советы

1. **Запускай ночью** — проверка 1000 слов займёт ~10 минут
2. **Сначала DRY_RUN** — проверь без исправлений
3. **Сделай бэкап БД** перед исправлениями
4. **Проверяй вручную** — ИИ может ошибаться

---

## 🚨 Ошибки

### `GROQ_API_KEY не найден`

Добавь в `.env.local`:
```env
GROQ_API_KEY=gsk_xxxxx
```

### `Rate limit exceeded`

Groq бесплатен с лимитами. Подожди минуту и запусти снова.

### `POSTGRES_URL не найдена`

Проверь `.env.local`:
```env
POSTGRES_URL=postgresql://...
```
