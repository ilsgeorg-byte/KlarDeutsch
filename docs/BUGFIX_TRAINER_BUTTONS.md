## ✅ Выполненные улучшения

### 1. Добавлено состояние `ratingInProgress`

```typescript
const [ratingInProgress, setRatingInProgress] = useState(false);
```

**Назначение:** Блокировка кнопок во время отправки оценки на сервер.

---

### 2. Улучшена функция `handleRate`

**Было:**
```typescript
const handleRate = async (rating: number) => {
  if (!currentWord) return;
  try {
    const res = await fetch("/api/trainer/rate", {...});
    if (res.ok) handleNext();
  } catch (err) {
    console.error(err); // ❌ Пользователь не видит ошибку
  }
};
```

**Стало:**
```typescript
const handleRate = async (rating: number) => {
  if (!currentWord || ratingInProgress) return; // ✅ Защита от повторных нажатий
  
  setRatingInProgress(true); // ✅ Начинаем загрузку
  try {
    const res = await fetch("/api/trainer/rate", {...});
    
    if (!res.ok) { // ✅ Обработка ошибок API
      const errorData = await res.json().catch(() => ({}));
      setAudioStatus(`Ошибка: ${errorData.error || 'Не удалось сохранить прогресс'}`);
      return;
    }
    
    handleNext(); // ✅ Переход к следующему слову
  } catch (err) {
    setAudioStatus("Ошибка сети. Проверьте подключение к серверу.");
  } finally {
    setRatingInProgress(false); // ✅ Завершаем загрузку
  }
};
```

---

### 3. Визуальная обратная связь на кнопках

**Добавлено:**
- ✅ Блокировка кнопок (`disabled={ratingInProgress}`)
- ✅ Снижение прозрачности (`opacity-50`)
- ✅ Курсор `cursor-not-allowed`
- ✅ Спиннер загрузки (`Loader2`)
- ✅ Убраны hover-эффекты во время загрузки

**Пример для кнопки "Сложно":**
```tsx
<button
  onClick={() => handleRate(1)}
  disabled={ratingInProgress}
  className={`...${
    ratingInProgress 
      ? 'opacity-50 cursor-not-allowed' 
      : 'hover:bg-red-50 hover:border-red-300 hover:shadow-md'
  }`}
>
  {ratingInProgress ? (
    <Loader2 size={20} className="animate-spin text-red-500 mb-1" />
  ) : (
    <span className="text-red-500 font-bold mb-1">Сложно</span>
  )}
  <span>Завтра</span>
</button>
```

---

### 4. Блокировка переворота карточки

**Было:**
```typescript
onClick={() => !showAnswer && setShowAnswer(true)}
```

**Стало:**
```typescript
onClick={() => !showAnswer && !ratingInProgress && setShowAnswer(true)}
```

**Назначение:** Предотвращение переворота карточки во время оценки.

## 🧪 Тестирование

### Сценарии для проверки:

1. **Нормальная работа**
   - ✅ Нажать на кнопку оценки
   - ✅ Появляется спиннер
   - ✅ Кнопки блокируются
   - ✅ Переход к следующему слову

2. **Ошибка API (сервер недоступен)**

   - ✅ Быстро нажать несколько раз
   - ✅ Срабатывает только первое нажатие
   - ✅ Остальные игнорируются
