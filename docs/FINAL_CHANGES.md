## 🚀 Как использовать

### Быстрое добавление (Админка)
```
1. /admin/words → "Добавить слово"
2. Ввести de + ru
3. Нажать "Заполнить с помощью ИИ"
4. ИИ заполнит: артикль, уровень, тему
5. Нажать "Добавить"
```

### Расширенное добавление (Профиль)
```
1. /profile/my-words → "Добавить"
2. Ввести de + ru
3. Нажать "Заполнить с помощью ИИ"
4. Выбрать пример из вариантов
5. Нажать "Добавить"
```

### Массовая загрузка (Профиль)
```
1. /profile/my-words → "Добавить"
2. Вкладка "Загрузить из файла"
3. Перетащить CSV/JSON
4. Дождаться результата
```

---

## 📝 Примечания

- Кнопка добавления слов теперь доступна **только** в админке и профиле
- ИИ заполняет: артикль, уровень, тему, формы глагола, примеры
- Массовая загрузка поддерживает до 500 слов за раз
- Страница "Словарь" теперь только для поиска и просмотра
