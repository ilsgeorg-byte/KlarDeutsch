# Обработка отсутствующего множественного числа

## Проблема
Не у всех существительных в немецком языке есть форма множественного числа (например, `das Geld`, `die Milch`, `der Mut`).

Ранее поле `plural` оставалось пустым (NULL/`''`), что создавало неопределённость:
- Неясно, не проверено ли слово
- Или проверено, но множественного числа нет

## Решение (Вариант B)

### 1. Маркер отсутствия множественного числа
- Если множественное число **есть**: записывается форма (например, `die Unterschriften`)
- Если множественного числа **нет**: записывается маркер `—` (прочерк)

### 2. Изменения в коде

#### `app/api/admin/check-words/route.ts`
- Обновлён промпт для AI:
  ```
  corrected_plural: ОБЯЗАТЕЛЬНО укажи форму множественного числа. 
  Если слово не имеет множественного числа, верни "—" (прочерк).
  ```
- Обновлена логика обработки:
  - Заполняет `plural` для существительных, если поле пустое
  - Записывает как формы множественного числа, так и маркер `—`

#### `app/components/WordCard.tsx`
- Обновлён отображение:
  - Если `plural === '—'`: показывает "Pl: **нет мн.ч.**" (серым цветом)
  - Иначе: показывает "Pl: **die Unterschriften**"

#### `app/dictionary/[id]/page.tsx`
- Аналогичное обновление для страницы словаря

#### `app/admin/words/page.tsx`
- Обновлён placeholder в форме: `"напр. die Häuser или — если нет мн.ч."`

### 3. Скрипты для массового заполнения

#### `tools/check_missing_forms.py` - Диагностика
```bash
python tools/check_missing_forms.py
```
Покажет:
- Общее количество существительных без `plural`
- Количество слов без `verb_forms` (глаголы + прилагательные)
- Примеры таких слов

#### `tools/smart_fill_missing_forms.py` - Умное заполнение через AI
```bash
# 1. Сначала в режиме просмотра (DRY_RUN = True)
python tools/smart_fill_missing_forms.py

# 2. Для исправления: измени DRY_RUN = False в скрипте
python tools/smart_fill_missing_forms.py
```

Скрипт:
- **AI определяет тип каждого слова** (существительное/глагол/прилагательное/фраза)
- Для **существительных**: заполняет `article` и `plural` (или `—` если нет мн.ч.)
- Для **глаголов**: заполняет `verb_forms` в формате "Infinitiv, Präteritum, Partizip II"
- Для **прилагательных**: пропускает (им не нужны эти формы)
- Для **фраз**: пропускает

### 4. Как использовать

1. **Проверить проблему:**
   ```bash
   python tools/check_missing_forms.py
   ```

2. **Запустить умное массовое исправление:**
   ```bash
   # Измени DRY_RUN = False в tools/smart_fill_missing_forms.py
   python tools/smart_fill_missing_forms.py
   ```

3. **Перезапустить AI проверку из админки:**
   - Сбрось статус проверки (кнопка "Сбросить")
   - Запусти "AI проверка слов" снова

### 5. Примеры отображения

| Значение в БД | Отображение в интерфейсе |
|---------------|--------------------------|
| `die Unterschriften` | Pl: **die Unterschriften** |
| `—` | Pl: *нет мн.ч.* (серым) |
| `''` или `NULL` | Не отображается |

### 6. Преимущества

✅ **Ясность**: видно, что слово проверено и не имеет множественного числа  
✅ **Автоматизация**: AI сам определяет и заполняет формы  
✅ **UX**: пользователь видит явное сообщение "нет мн.ч." вместо пустоты  
✅ **Статистика**: можно точно посчитать, сколько слов не имеют множественного числа
