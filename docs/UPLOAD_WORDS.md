# 📥 Загрузка личных слов

## Возможности

Теперь вы можете добавлять свои собственные слова для изучения двумя способами:
1. **По одному** - через форму на странице
2. **Массово** - загрузка из файла (CSV или JSON)

---

## 📍 Где найти

1. Откройте **Профиль** → нажмите кнопку **"Мои слова"**
2. Или напрямую: `/profile/my-words`

---

## ➕ Добавление одного слова

1. Нажмите кнопку **"Добавить"** на странице "Мои слова"
2. Откроется модальное окно с двумя вкладками:
   - **"Одно слово"** - для добавления по одному
   - **"Загрузить из файла"** - для массовой загрузки

3. Заполните обязательные поля:
   - **Немецкое слово** (de) *
   - **Перевод** (ru) *
   
4. Дополнительные поля (необязательно):
   - **Артикль** - der, die, das или без артикля
   - **Уровень** - A1, A2, B1, B2, C1 (по умолчанию A1)
   - **Тема** - например, "Глаголы", "Существительные"
   - **Формы глагола** - для глаголов (например: `machen, machte, hat gemacht`)
   - **Пример использования** - предложение на немецком и русском

5. Нажмите **"Добавить слово"**

---

## 📁 Массовая загрузка из файла

### Поддерживаемые форматы

- **CSV** - текстовый файл с разделителями запятой
- **JSON** - структурированный формат данных

### Формат CSV

Создайте файл с расширением `.csv` и следующей структурой:

```csv
de,ru,article,level,topic,verb_forms,example_de,example_ru
machen,делать,,A1,Глаголы,,Ich mache meine Hausaufgaben.,Я делаю свою домашнюю работу.
gehen,идти,,A1,Глаголы,"gehen, ging, ist gegangen",Er geht zur Schule.,Он идёт в школу.
das Haus,дом,das,A1,Существительные,,Das Haus ist groß.,Дом большой.
```

**Поля:**
- `de` - немецкое слово (обязательно)
- `ru` - перевод (обязательно)
- `article` - артикль (der/die/das, необязательно)
- `level` - уровень (A1/A2/B1/B2/C1, необязательно, по умолчанию A1)
- `topic` - тема (необязательно, по умолчанию "Импорт")
- `verb_forms` - формы глагола (необязательно)
- `example_de` - пример на немецком (необязательно)
- `example_ru` - пример на русском (необязательно)

### Формат JSON

Создайте файл с расширением `.json`:

```json
{
  "words": [
    {
      "de": "verstehen",
      "ru": "понимать",
      "article": "",
      "level": "A1",
      "topic": "Глаголы",
      "verb_forms": "",
      "example_de": "Ich verstehe das nicht.",
      "example_ru": "Я этого не понимаю."
    },
    {
      "de": "der Apfel",
      "ru": "яблоко",
      "article": "der",
      "level": "A1",
      "topic": "Еда"
    }
  ]
}
```

**Альтернативный формат** (массив слов):

```json
[
  {
    "de": "machen",
    "ru": "делать",
    "level": "A1"
  },
  {
    "german": "gehen",
    "russian": "идти",
    "level": "A1"
  }
]
```

**Поддерживаемые варианты ключей:**
- Немецкое слово: `de`, `german`, `word`
- Перевод: `ru`, `russian`, `translation`
- Артикль: `article`
- Уровень: `level`
- Тема: `topic`
- Формы глагола: `verb_forms`, `forms`
- Пример немецкий: `example_de`, `example`
- Пример русский: `example_ru`, `example_translation`

### Как загрузить файл

1. Откройте вкладку **"Загрузить из файла"**
2. Перетащите файл в область загрузки или нажмите для выбора файла
3. Дождитесь завершения загрузки
4. Просмотрите результат:
   - ✅ **Добавлено** - успешно загруженные слова
   - ⚠️ **Пропущено** - слова, которые уже существуют или имеют ошибки

**Ограничения:**
- Максимум **500 слов** за одну загрузку
- Поддерживаемые файлы: `.csv`, `.json`
- Кодировка: **UTF-8**

## ❗ Возможные ошибки

| Ошибка | Причина | Решение |
|--------|---------|---------|
| "Список слов пуст" | Файл не содержит слов | Проверьте формат файла |
| "Неверный формат JSON" | Ошибка в JSON структуре | Проверьте синтаксис JSON |
| "Максимум 500 слов" | Превышен лимит | Разбейте файл на несколько частей |
| "Слово уже существует" | Такое слово уже есть в базе | Пропущено автоматически |
| "Нет de или ru" | Отсутствует немецкое слово или перевод | Добавьте обязательные поля |

## 🗑️ Удаление слов

На странице "Мои слова" вы можете удалять свои слова:
1. Нажмите кнопку 🗑️ **Корзина** рядом со словом
2. Подтвердите удаление

**Примечание:** Можно удалять только свои личные слова. Общие слова базы доступны только для чтения.

---

## 💡 Советы

1. **Группируйте слова по темам** - указывайте тему при загрузке (например, "Глаголы движения", "Еда", "Путешествия")

2. **Используйте уровни** - распределяйте слова по уровням сложности (A1-C1)

3. **Добавляйте примеры** - слова с примерами использования запоминаются лучше

4. **Для глаголов указывайте формы** - это поможет при изучении спряжения

5. **Регулярно делайте импорт** - сохраняйте новые слова в файл и загружайте пакетом

## 🔧 API Endpoints

Для разработчиков доступны следующие endpoint'ы:

```
POST   /api/words/custom          - Добавить одно слово
POST   /api/words/bulk-upload     - Массовая загрузка слов
GET    /api/words/my-words        - Получить список личных слов
DELETE /api/words/my-words?id=X   - Удалить личное слово
```

**Пример запроса (JSON):**
```bash
curl -X POST http://localhost:5000/api/words/bulk-upload \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "words": [
      {"de": "machen", "ru": "делать", "level": "A1"}
    ]
  }'
```
