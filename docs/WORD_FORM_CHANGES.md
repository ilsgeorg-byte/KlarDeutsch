# 📝 Изменения в функционале добавления слов

## Обзор изменений

Функционал добавления слов был разделён на два уровня:

### 1. 📖 Словарь - Быстрое добавление
**Страница:** `/dictionary`  
**Компонент:** `AddWordModal.tsx`

**Поля формы:**
- Немецкое слово *
- Перевод *
- Артикль (необязательно)
- Уровень (необязательно)

**Особенности:**
- Минималистичная форма для быстрого добавления
- Кнопка "Добавить слово"
- Подсказка о переходе в профиль для расширенного добавления

---

### 2. 👤 Профиль - Расширенное добавление
**Страница:** `/profile/my-words`  
**Компонент:** `UploadWordsModal.tsx`

**Вкладка "Одно слово":**
- Немецкое слово *
- Перевод *
- **Кнопка "Заполнить с помощью ИИ"** ✨
- Артикль
- Уровень
- Тема
- Формы глагола
- Примеры использования (выбор из AI или вручную)

**Вкладка "Загрузить из файла":**
- Drag & Drop зона
- Поддержка CSV и JSON
- Массовая загрузка до 500 слов
- Отчёт о загрузке

## 🚀 Использование

### Быстрое добавление (Словарь)
```
1. Открыть /dictionary
2. Нажать "Добавить слово"
3. Ввести de + ru
4. Нажать "Добавить"
```

### Расширенное добавление (Профиль)
```
1. Открыть /profile/my-words
2. Нажать "Добавить"
3. Ввести de + ru
4. Нажать "Заполнить с помощью ИИ"
5. Выбрать пример (если нужно)
6. Нажать "Добавить"
```

### Массовая загрузка
```
1. Открыть /profile/my-words
2. Нажать "Добавить"
3. Выбрать вкладку "Загрузить из файла"
4. Перетащить CSV/JSON файл
5. Просмотреть результат
```
