# KlarDeutsch - Общая документация

## Обзор

**KlarDeutsch** — это комплексное веб-приложение для изучения немецкого языка, разработанное преимущественно для русскоязычных пользователей. Платформа сочетает карточки с интервальным повторением, коррекцию текста на основе ИИ и интерактивную тренировку словарного запаса, делая изучение немецкого эффективным и увлекательным.

**Девиз:** "Учи немецкий легко"  
**Миссия:** Твой персональный помощник для уровня A1 и выше

---

## 🌟 Основные возможности

### 1. **Интерактивный тренажёр** (`/trainer`)
- Система интервального повторения на основе **алгоритма SM-2** (SuperMemo 2)
- Практика словарного запаса с помощью карточек
- Пользователи оценивают свои знания: 0 (Не знаю), 1 (Трудно), 3 (Хорошо), 5 (Легко)
- Интеллектуальное планирование с fuzzing и корректировкой фактора лёгкости
- Отслеживает прогресс обучения и оптимизирует интервалы повторений

### 2. **Словарь** (`/dictionary`)
- Просмотр словарного запаса по **уровням CEFR** (A1, A2, B1, B2, C1)
- Организация по тематическим темам
- Расширенный поиск
- Каждая словарная статья включает:
  - Определённый артикль (der/die/das)
  - Формы множественного числа
  - Спряжение глаголов (если применимо)
  - Примеры предложений
  - Синонимы и антонимы
  - Распространённые коллокации

### 3. **Дневник с ИИ-коррекцией** (`/diary`)
- Свободное написание текстов на немецком
- **Коррекция грамматики на основе ИИ** через Google Gemini или OpenAI
- Подробные объяснения каждой корректировки
- Извлечение лексики из ваших записей для добавления в словарь

### 4. **Тренировка произношения** (`/audio`)
- Записывайте произношение прямо в браузере
- Сохраняйте аудиофайлы (форматы webm, mp3, wav, ogg)
- Сравнивайте своё произношение с носителями языка

### 5. **Тематические темы** (`/topics`)
- Словарный запас, организованный по жизненным ситуациям
- Изучайте слова в контексте (например, "В ресторане", "Путешествие", "Покупки")
- Структурированные пути обучения

### 6. **Система избранного**
- Закладки важных слов для быстрого доступа
- Управление личным списком слов

### 7. **Профиль пользователя** (`/profile`)
- Отслеживание статистики обучения
- Просмотр метрик прогресса
- Мониторинг освоенных слов

### 8. **Панель администратора** (`/admin`)
- Доступ с защитой JWT со списком email
- Управление словами (CRUD операции)
- Управление пользователями
- Контроль словаря
- Панель статистики
- Инструменты обогащения слов на основе ИИ

## 🛠️ Технологический стек

### Фронтенд
- **Next.js 14** - React фреймворк с App Router
- **React 18.2** - Библиотека UI
- **TypeScript 5.3.3** - Типобезопасная разработка (строгий режим)
- **Tailwind CSS 3.4** - Утилитарный CSS фреймворк с поддержкой тёмного режима
- **SWR** - Получение данных и кэширование
- **Lucide React** - Библиотека иконок
- **Шрифт Google Inter** с кириллической подмножеством

### Бэкенд
- **Flask 3.0** - Python веб-фреймворк
- **PostgreSQL 15** - Основная база данных (через psycopg2 с пулом соединений)
- **Redis 5.0** - Слой кэширования
- **Pydantic 2.6** - Валидация запросов/ответов
- **PyJWT** - Аутентификация
- **bcrypt** - Хеширование паролей
- **bleach** - Санитизация HTML
- **Google Generative AI (Gemini)** - Коррекция текста ИИ
- **OpenAI** - Альтернативный ИИ провайдер
- **Gunicorn 21.2** - Production WSGI сервер

### Инфраструктура
- **Docker** - Многоэтапные сборки
- **Docker Compose** - Оркестрация 5 сервисов
- **Nginx** - Обратный прокси
- **Vercel** - Поддержка развёртывания

## 🚀 Начало работы

### Предварительные требования
- Node.js 20+
- Python 3.10+
- PostgreSQL 15+
- Redis 5.0+

### Быстрый запуск с Docker

1. **Клонируйте репозиторий:**
   ```bash
   git clone <repository-url>
   cd KlarDeutsch
   ```

2. **Настройте переменные окружения:**
   ```bash
   cp .env.local.example .env.local
   # Отредактируйте .env.local с вашей конфигурацией
   ```

3. **Запустите все сервисы:**
   ```bash
   docker-compose up -d
   ```

4. **Откройте приложение:**
   - Фронтенд: http://localhost
   - Backend API: http://localhost:5000

### Ручная установка

1. **Фронтенд:**
   ```bash
   npm install
   npm run dev
   ```

2. **Бэкенд:**
   ```bash
   pip install -r requirements.txt
   python api/app.py
   ```

## 🏗️ Архитектура

### Алгоритм интервального повторения SM-2

Платформа использует проверенный алгоритм SuperMemo-2 для оптимального обучения:

- **Фактор лёгкости (EF):** Начинается с 2.5, корректируется на основе результатов
- **Расчёт интервала:** 
  - Q < 2: Сброс на день 1
  - Q >= 2: Интервал = предыдущий интервал × EF
- **Fuzzing:** Добавляет случайность для предотвращения массовых повторений
- **Отслеживание прогресса:** Каждое слово поддерживает своё расписание повторений

### Обогащение слов ИИ

Слова обогащаются данными, сгенерированными ИИ, следуя строгим лингвистическим правилам:

- **Существительные:** Всегда имеют артикли и формы множественного числа
- **Глаголы:** Никогда не имеют артиклей, включают спряжения
- **Прилагательные:** Включают сравнительную и превосходную степени
- **Все слова:** Примеры, синонимы, антонимы, коллокации, классификация по темам

Смотрите `GEMINI.md` для детальных лингвистических указаний.

### Функции безопасности

- Аутентификация на основе JWT с HTTP-only cookies
- Белый список email для доступа администратора
- Санитизация входных данных (bleach)
- Защита от SQL-инъекций (параметризованные запросы)
- Конфигурация CORS
- Заголовки безопасности (HSTS, X-Frame-Options и т.д.)
- Ограничение частоты запросов API
- Хеширование паролей с bcrypt

## 📊 Схема базы данных

Ключевые таблицы:
- **users:** Учётные записи и профили пользователей
